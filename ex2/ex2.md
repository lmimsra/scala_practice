# ex2のファイル
## 色々な型  
### 文字型
|type |detail |
|:---------|:---------|  
| Byte | １バイト符号付き整数 |
| Short | 2バイト符号付き整数 |
| Int | 4バイト符号付き整数 |
| Long | 8バイト符号付き整数 |
| Char | 2バイト符号なしUnicode文字 |
| String | charの連続した値 |
| Float | 4バイト単精度浮動小数点数 |
| Double | 8バイト単精度浮動小数点数 |
| Boolean | true or false |

### エスケープ文字  

|type |detail |
|:---------|:---------|
| ￥n | 改行 |
| ￥b | バックスペース |
| ￥t | タブ |
| ￥f | 改ページ |
| ￥r | 復帰 |
| ￥' | シングルクォート |
| ￥" | ダブルクォート |
| ￥￥ | 円記号 |

### 文字列
表示の違い  
```
  "文字列"　→　文字列
  """ '文字列'  "文字列" """
  ↓
  '文字列' "文字列"
```
